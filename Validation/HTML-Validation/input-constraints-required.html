Input Constraints w/ required

Validating <input> with HTML5

Ex. Choose your favorite junk food. There are two choices: pretzels or chips. There will be a text <input> with a matching label, and a submit <button>

The required attribute

  • simplest to use
  • creates a mandatory input field
  • can mark the element using this attribute
  • when set, the form won't submit and will display an error
  • if input is empty this will be considered invalid
