Form Validation w/ Javasctipt

Javascript can be used to style native error messages. Some browsers don't yet support the built-in form validation of HTML.

HTML5 Constraint Validation API

This API contains methods and properties that can be used on each form element.

For a more exhaustive look at this API look at:

MDN Guide on Constraint Validation API
