The <form> element

The <form>; is used to create HTML forms.x

The <fieldset> and <legend> elements

The <fieldset> is used to create a group of widgets that share the same purpose. The <fieldset> can be labeled by including a <legend> element just below the opening <fieldset> tag. The text content of the <legend> element formallu describes the purpose of the <fieldset> element it is inside.

The <label> element

The <label> element is important if you want to build accessible web forms. Screenreaders can speak a form element's label along with instructions.

The screen reader will output something like "Name, edit text", for this example.

Clickable Lables

Properly set up labels can be clickable.

Multiple Labels

Multiple labels can be put in a single widget. However this does not work with some assistive technologies. In the case of multiple labels, the widget and it's labels should be nested in a single <label> element
