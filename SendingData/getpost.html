GET and POST methods

GET method EXAMPLE:

When the form is sent, by clicking the button, this: https://www.google.com/?say=Greetings&to=Person is what appears in the address bar. The data is added to the URL as a series of name/value pairs.

The HTTP request looks like this:

GET /?say=Greetings&to=Person HTTP/1.1
Host: google.com

POST method EXAMPLE:

With the POST method, the browser talks to a server and asks for a response that takes into account the data provided in the body of the HTTP request.

"Hey server, take a look at this data
and send me back an appropriate response."

When the POST method is used te data is appended to the body of the HTTP request.

** In this case you get an error, because the URL does not allow the use of POST **

This is what the HTTP request looks like:
POST / HTTP/1.1
Host: google.com
Content-Type: application/x-www-form-urlencoded
Content-Length: 19
Say=Hello&to=Person

Visit https://developer.mozilla.org/en-US/docs/Learn/HTML/Forms/Sending_and_retrieving_form_data
for more on security and how different frameworks use GET and POST
