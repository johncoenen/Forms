Creating a custom <select> widget

    Rebuilding

  • Start with an existing widget(<select>)
  • Try to mimic something that looks like what you want.

Our Widget States

Normal:

  • on page load
  • when user clicks outside widget during active state, returns to normal
  • the user moves focus to another widget

Active

  • the user clicks on it
  • the user hits the tab key and it gains focus
  • the widget is in its open state and the user clicks on the widget

Open

  • the widget is in any other state than open and the user clicks on it

Our Widget Values

It is important to define how the values in the widget change.

The value changes when:

  • The user clicks on an option during it's open state
  • The user hits up/down arrow keys when eidget is in active state

Widget Option Behavior

  • when widget is open, the selection will be highlighted
  • mouse over an option results in the item becoming highlighted and previously highlighted item returns to normal state
