Advanced Form Widgets:

Notes and Examples

Numbers

  • created with the <input> element, with its type attribute set to the value number

  • Looks like a text field but only allows numbers. Depending on the browser buttons to increase and decrease the number value may be provided.

  • Can set min and max on the value attribute

  • Specify incremental amounts the increase and decrease buttons change using the step attribute

Example



Sliders

  • Sliders can also be used to pick numbers.

  • Sliders are, visually, less accurate than text fields. The numbers exact value is less important.

  • created using the <input> with its type attribute set to the value range

  • To set the slider to be more accurate include min, max, and step

Date/Time Picker

  • The date and time control is created using the <input> element and a value for the type attribute. This value depends on if you want just date, or just time, or both.

datetime-local

This creates a widget to display a date and time without any time zone information.

Example

month

This creates a widget to display and pick a month with the year.

Example

time

This creates a widget to display and pick a time value.

Color Picker

There are different ways to show color pickers(RGB, decimal, hexidecimal, HSL, values, keywords, etc.

The color picker uses both textual and visual elements to show the user what they are choosing.

The color picker is created using the <input> element with its type set to the value color.
